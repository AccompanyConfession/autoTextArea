# autoTextArea文本域高度自适应插件 @陪白
autoTextArea是一个简单的基于jquery的文本域高度自适应插件


- 官网： http://accompanyconfession.online（作者）
##
### 说明
- 在html页面中引入autoTextarea.min.js,前提是已经引入了jquery
- 在主页添加$("#元素选择").autoTextarea();完成初始化。
- $("元素选择").autoTextarea();中的元素选择可以为id选择器,类选择器,标签选择器。
### 参数
- maxHeight：最大的高度(默认500)。
- extra：初始化时的光标与输入框保持的距离(默认0)。
- message：文本框的内容。
